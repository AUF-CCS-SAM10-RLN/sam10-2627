How to use these references

Start with the practical books for hands-on skill building, then use the papers and standards to strengthen conceptual understanding and technical justification during labs and project presentations.

Books

Recommended core books

Practical reading for Linux systems administration, AWS operations, and reliability-oriented thinking.

UNIX and Linux System Administration Handbook

  • Why it is useful: broad, durable systems administration reference covering Linux and operational practice.
  • Best fit: Linux administration, services, storage, operations, and maintenance topics.
  • Publisher page

How Linux Works, 3rd Edition

  • Why it is useful: strong conceptual explanation of how Linux boots, runs processes, manages storage, and handles networking.
  • Best fit: Linux architecture, shell environment, services, and file system topics.
  • Publisher page

Linux Pocket Guide, 4th Edition

  • Why it is useful: compact command-line reference for daily administration practice.
  • Best fit: CLI command mastery and lab support.
  • Publisher page

AWS System Administration

  • Why it is useful: focused on administering AWS resources from a systems administrator perspective.
  • Best fit: IAM, security groups, EC2, networking, and operational cloud workflows.
  • Publisher page

AWS for System Administrators

  • Why it is useful: more modern AWS-administration-oriented practical text with infrastructure, IAM, and automation coverage.
  • Best fit: compute, IAM, CloudFormation, and cloud operations topics.
  • Publisher page

Site Reliability Engineering / The Site Reliability Workbook

  • Why it is useful: strong operational thinking for monitoring, reliability, toil reduction, and production service management.
  • Best fit: monitoring, performance, reliability, maintenance, and optimization topics.
  • Google SRE library

Research manuscripts

Foundational papers worth reading

Seminal references for cloud computing, distributed systems, and systems administration theory.

The NIST Definition of Cloud Computing

  • Why it matters: standard vocabulary for cloud models and service models.
  • Best fit: Topics 1 and 2.
  • NIST publication

Above the Clouds: A Berkeley View of Cloud Computing

  • Why it matters: classic early analysis of cloud computing economics and architecture.
  • Best fit: cloud computing fundamentals and infrastructure trade-offs.
  • USENIX page

Dynamo: Amazon’s Highly Available Key-value Store

  • Why it matters: key distributed-systems paper from Amazon that helps explain NoSQL design trade-offs.
  • Best fit: database administration, availability, scalability, CAP-related discussions.
  • Amazon Science page

MapReduce: Simplified Data Processing on Large Clusters

  • Why it matters: foundational distributed processing paper from Google.
  • Best fit: distributed systems and cloud-scale processing discussion.
  • Google Research page

On the Theory of System Administration

  • Why it matters: connects systems administration work to formal thinking about policy, automation, and maintenance.
  • Best fit: Topic 1 and operations-oriented later topics.
  • arXiv manuscript

Technical articles and standards

High-value official references

Current official guidance that supports secure administration, operations, and architecture review.

AWS Well-Architected Framework

  • Why it matters: current AWS architecture best-practice framework for reliability, security, performance, cost, and sustainability.
  • Best fit: Topic 15 and any architecture or optimization discussion.
  • Official AWS documentation

IT System and Network Administration (NIST)

  • Why it matters: a gateway to official NIST resources for secure systems and network administration.
  • Best fit: operations, maintenance, hardening, and governance topics.
  • Official NIST page

Google SRE Resources Hub

  • Why it matters: reliable source for production operations, SLOs, incident thinking, and service reliability guidance.
  • Best fit: monitoring, reliability engineering, maintenance, and troubleshooting topics.
  • Official Google SRE resources
