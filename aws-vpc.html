Security review

  • Only the ALB and bastion expose public routes.
  • Private workloads use outbound-only internet access through NAT.
  • Database support traffic is restricted to application and admin sources.

Learning note

Student focus

  • Distinguish public from private subnets by route target, not name alone.
  • Relate security groups to instance-level policy and NACLs to subnet policy.
  • Map VPC components to traditional networking concepts.
