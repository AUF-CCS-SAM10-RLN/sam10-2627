Course description

What the course covers

This course introduces the fundamental concepts and practices of systems administration and maintenance using Amazon Web Services (AWS) cloud platform. It provides students with the knowledge and practical skills needed to deploy, configure, manage, secure, monitor, and maintain cloud-based systems through lectures and hands-on laboratory activities. Students will learn how to configure and manage AWS resources while applying standard systems administration practices. Students will be able to deploy and maintain a functional cloud environment and apply systems administration principles to support real-world IT operations.

Major course outcomes

Program-level deliverables

MCO1: AWS Infrastructure Deployment

Students will design and deploy a basic cloud infrastructure on AWS by configuring virtual networks, virtual machines, storage, and user access. They will document the system architecture and demonstrate that the deployed environment is functional, secure, and properly configured.

MCO2: Systems Administration and Maintenance

Students will administer and maintain their AWS environment by implementing monitoring, backups, security measures, and basic automation to ensure reliable system operation. They will demonstrate their ability to troubleshoot common system issues and present an operations report describing the maintenance procedures and improvements made to their cloud infrastructure.

Course learning outcomes

CLO alignment overview

  • Explain the fundamental concepts of systems administration, cloud computing, and AWS cloud infrastructure, including core cloud services and security principles.
  • Deploy and configure cloud computing resources, including networking, compute, storage, identity management, and database services using AWS, leveraging AI-assisted tools where appropriate while validating configurations against established best practices.
  • Administer and maintain Linux-based cloud systems by managing users, services, storage, web applications, and databases using standard systems administration practices, utilizing AI to support automation, scripting, and technical documentation.
  • Monitor, secure, troubleshoot, and optimize cloud-based systems by applying appropriate maintenance practices and AWS architectural best practices, and AI-assisted analysis while exercising professional judgment to verify recommendations and implement appropriate solutions.

Topic 1

Systems theory, IT infrastructure, operating systems, client-server architecture, and AWS foundations.

Topic 2

Cloud models, virtualization, distributed systems, AWS compute, and pricing concepts.

Topic 3

Identity, access control, AWS IAM, MFA, federation, and governance practices.

Topic 4

Networking models, DNS, routing, VPCs, subnets, gateways, and cloud network security.

Topic 5

EC2, AMIs, instance types, launch templates, Elastic IP, provisioning, and CloudFormation basics.

Topic 6

Linux architecture, shell environment, file systems, permissions, process management, SSH, and Systems Manager.

Topic 7

Storage hierarchy, object, block, and file storage, RAID, performance, and AWS storage services.

Topic 8

Relational and NoSQL databases, ACID, CAP theorem, RDS, Aurora, DynamoDB, and database backup.

Topic 9

Multi-tier architecture, web servers, reverse proxy, load balancing, Route 53, ACM, and deployment.

Topic 10

Monitoring, observability, performance metrics, CloudWatch, CloudTrail, AWS Config, and SNS.

Topic 11

Cloud security principles, defense in depth, KMS, Secrets Manager, GuardDuty, Inspector, and WAF.

Topic 12

Reliability engineering, scalability, fault tolerance, elasticity, Auto Scaling, ELB, Multi-AZ, and health checks.

Topic 13

Business continuity, disaster recovery, RPO, RTO, backups, AWS Backup, snapshots, replication, and versioning.

Topic 14

Systems maintenance, IT service management, automation, patching, root cause analysis, and AWS operational tooling.

Topic 15

AWS Well-Architected pillars, quality attributes, infrastructure optimization, Cost Explorer, and Trusted Advisor.
