Interactive Practice

Linux Bash practice shell

Students can practice safe Bash commands in a realistic terminal-style environment. The shell is static and course-controlled, so it works on GitHub Pages while still reinforcing command usage from the labs.

Command focus

What this shell supports

  • `pwd`, `ls`, `cd`, `mkdir`, `touch`, `cp`, `mv`, `cat`, `grep`, `clear`
  • `whoami`, `id`, `groups`, `uname -a`, `hostnamectl`
  • `ps aux`, `top`, `free -h`, `df -h`, `lsblk`
  • `ip a`, `ip r`, `ss -tuln`, `dig`, `curl`, `ping`, `traceroute`
  • `systemctl status webapp`, `journalctl -u webapp`, `chmod`, `sudo`, `ssh`

Goal

Network inspection

Goal

Inspect addresses, routes, name resolution, and service reachability.

Logs and troubleshooting

Goal

Read service logs and search for operational issues.

Try

  • journalctl -u webapp
  • grep ERROR webapp.log
  • cat /etc/os-release
  • ssh admin@web-01
